# WanderLust

WanderLust is a full-stack web application inspired by Airbnb, built using the MERN stack. This project was collaboratively developed to allow users to explore, create, and manage property listings through an intuitive and user-friendly interface.

## Features

- Create, view, update, and delete property listings  
- User authentication with secure login, signup, and logout functionality  
- Add and manage reviews for listings  
- Interactive maps powered by Mapbox to visualize property locations  
- Responsive design for a seamless user experience  

## Tech Stack

- Frontend: HTML, CSS, JavaScript, EJS  
- Backend: Node.js, Express.js  
- Database: MongoDB (Mongoose)  
- Authentication: Passport.js  
- Maps Integration: Mapbox API  

## Description

The application follows a RESTful architecture and efficiently manages data for users, listings, and reviews using MongoDB. It provides a dynamic platform where users can explore different locations and share their own listings.

## Collaboration

This project was developed as a team effort using GitHub for version control. Both contributors worked collaboratively by managing code through commits, pull-push workflows, and shared development responsibilities.
